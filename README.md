# AutoLoginPacketBrute

基于HTTP报文进行弱口令动态爆破

为了补充对于常规登录报文需要额外配置的盲点

实现快速对于常规场景进行登录爆破


## 功能说明

输入HTTP报文（自动定位|手动定位） 用户名和密码参数

读取账号密码字典 进行 循环替换和多线程报文重放
